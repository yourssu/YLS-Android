YLS

class YLS

Yourssu Logging System, inspired by Timber

// 사용 예
// Application의 onCreate()에서 초기화
YLS.init(
     platform = "android",
     user = YLS.generateRandomId(10), // 10자리 랜덤 문자열
     logger = YLS.RemoteLogger() // Logger 객체 지정
)

// init 때 지정한 Logger 객체를 사용하여 로깅
YLS.log("event" to "click")
